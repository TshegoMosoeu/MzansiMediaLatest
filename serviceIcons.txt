3️⃣ What does % mean?

% is the modulus operator.

It returns the remainder after division.

Example:

5 % 3  // = 2
4 % 3  // = 1
3 % 3  // = 0
4️⃣ What is index % serviceIcons.length doing?

It ensures the index wraps around the icon array.

Let’s say:

serviceIcons.length === 3

So:

index	index % 3	Icon Selected
0	0	icon 1
1	1	icon 2
2	2	icon 3
3	0	icon 1 again
4	1	icon 2 again
5	2	icon 3 again

So instead of crashing when index goes beyond the array length, it cycles back to the start.

Reuses icons cleanly

Avoids "undefined" errors

Keeps visual variety

Without this, if you did:

Which direction?

Why are we using modulo

Great question. The reason we use modulo (%) here is to make the index loop safely inside the bounds of an array.

Let’s break it down clearly.

The Problem

You have:

const serviceIcons = [Icon1, Icon2, Icon3]

That means:

serviceIcons.length === 3

Now imagine you’re mapping 6 services:

services.map((service, index) => { ... })

Indexes will be:

0, 1, 2, 3, 4, 5

But your icons array only has indexes:

0, 1, 2

So what happens if you do this?

serviceIcons[index]

At:

index = 3 → undefined ❌

index = 4 → undefined ❌

index = 5 → undefined ❌

That would break your UI.

What Modulo Does
index % serviceIcons.length

Modulo gives the remainder after division.

If length is 3:

index	index % 3	Result
0	0	safe
1	1	safe
2	2	safe
3	0	wraps back
4	1	wraps back
5	2	wraps back

So it cycles:

0 → 1 → 2 → 0 → 1 → 2 → ...
Why We Use It

We use modulo to:

✅ Prevent “undefined” errors
✅ Reuse a smaller array repeatedly
✅ Create a repeating pattern
✅ Keep indexes inside array bounds
Conceptually

Modulo turns a growing number into a repeating pattern.

Without modulo:

0 1 2 3 4 5 6 7

With % 3:

0 1 2 0 1 2 0 1
In UI Design

This is commonly used for:

Rotating icons

Alternating layouts (left/right)

Cycling background colors

Patterned grids

Example:

const layout = index % 2 === 0 ? "left" : "right"

This alternates layout every row.

Simple Real-World Analogy

Think of it like days of the week.

If today is day 10, and there are 7 days in a week:

10 % 7 = 3

So it’s the 3rd day of the cycle.

Modulo makes things repeat.

Summary

We use modulo because:

It keeps the index safely within the size of the array by looping back to the start when it reaches the end.
